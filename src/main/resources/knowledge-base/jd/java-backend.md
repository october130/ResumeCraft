# Java 后端开发工程师 JD

## 岗位要求
- 本科及以上学历，计算机相关专业
- 1-3 年 Java 后端开发经验
- 熟练掌握 Java 基础，包括集合、并发、JVM 原理
- 熟练掌握 Spring Boot、Spring MVC、MyBatis 等主流框架
- 熟悉 MySQL、Redis，了解数据库优化和缓存策略
- 熟悉 Linux 常用命令，能独立排查问题
- 了解微服务架构，有 Spring Cloud 经验者优先
- 有良好的编码习惯和文档意识

## 技能要求
- Java 基础：集合框架、多线程、JVM 调优
- 框架：Spring Boot、MyBatis、Spring MVC
- 数据库：MySQL（索引优化、事务）、Redis（缓存、分布式锁）
- 中间件：RabbitMQ / Kafka（消息队列）、Elasticsearch（搜索）
- 工具：Git、Maven、Docker、Linux

## 加分项
- 有分布式系统开发经验
- 熟悉微服务架构（Spring Cloud、Dubbo）
- 有高并发、高可用系统设计经验
- 有开源项目贡献经验
